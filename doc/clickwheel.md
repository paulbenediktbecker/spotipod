# Clickwheel

## install

### .net
sdk:
```
sudo apt-get update && \
  sudo apt-get install -y aspnetcore-runtime-8.0
```
runtime: 
```
sudo apt-get update && \
  sudo apt-get install -y aspnetcore-runtime-8.0
```

### install Clickwheel dotnet and nuget
```
dotnet nuget add source Clickwheel
```


### mono-compolete
```
sudo apt install mono-complete
```
